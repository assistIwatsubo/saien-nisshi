# 菜縁日誌 - バックエンド (api)

本プロジェクトのバックエンド（API層）リポジトリです。
チーム開発を意識して、compose.yamlの時点からDocker環境内でのセットアップにこだわりました。


## 🚀 開発のポイント

- Docker、Docker Composeを利用し、開発環境依存を可能な限り軽減。
- 達人に学ぶDB設計指南書第二版を読了し、draw.ioでER図を作成。その通りにテーブル作成を行いました。
- Seederなど、Laravelの便利機能も活用。
- 別リポジトリのLaravel APIと連携し、データ設計・テーブル設計などを練習。データの取得やユーザー認証も行っています。

## 🛠 使用技術

- **Language/Framework**: PHP 8.4 / Laravel 12
- **Infrastructure**: Docker / Docker Compose
- **Database**: MySQL 8.3
- **Tools**: draw.io
